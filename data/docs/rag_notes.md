# RAG Notes

RAG significa Retrieval-Augmented Generation.

## Ideia central

O sistema primeiro busca trechos relevantes nos documentos e depois usa esses trechos para responder.

## Vantagens

- reduz respostas inventadas
- melhora o uso de conhecimento local
- permite trabalhar com documentacao interna

## Exemplo

Pergunta:

```text
O que e RAG?
```

Resposta esperada:

```text
RAG e uma abordagem em que o modelo consulta documentos relevantes antes de responder.
```
